## Instalación :inbox_tray:

```bash

> git clone https://github.com/mendoza/toolbox-challenge

>  cd  toolbox-challenge

> cd API

> npm install

> cd ../front

> npm install

```

## Correr modo producción :running:

```bash

> docker-compose up

```


## Correr modo desarrollo :man_technologist:

```bash

# terminal 1
> cd API
> npm run dev

# terminal 2
> cd front
> npm start
```
